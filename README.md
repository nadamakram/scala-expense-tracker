# Scala Expense Tracker

A simple console-based **Expense Tracker** application written in Scala.  
It helps you keep track of your expenses, categorize them, and calculate totals. Perfect for learning Scala basics like **case classes**, **collections**, and **pattern matching**.

---

## Features

- Add and store expenses with categories
- Calculate total expenses
- Filter expenses by category
- Demonstrates Scala concepts:
  - `case class` and `sealed trait`
  - Immutable collections
  - Companion objects
